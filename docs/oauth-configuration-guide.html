Google OAuth Configuration

Active
1

Create a Google Cloud Project

Visit the Google Cloud Console and create a new project or select an existing one.

2

Enable Google+ API

Navigate to "APIs & Services" ‚Üí "Library" and search for "Google+ API". Click on it and press "Enable".

3

Create OAuth 2.0 Credentials

Go to "APIs & Services" ‚Üí "Credentials" ‚Üí "Create Credentials" ‚Üí "OAuth client ID"

  • Application type: Web application
  • Name: WaveMAX Affiliate Program
  • Authorized JavaScript origins:
    https://wavemax.promo https://www.wavemax.promo
  • Authorized redirect URIs:
    https://wavemax.promo/api/v1/auth/google/callback
4

Configure OAuth Consent Screen

Set up your OAuth consent screen with the following:

  • App name: WaveMAX Affiliate Program
  • User support email: support@wavemax.promo
  • App logo: Upload WaveMAX logo
  • Application home page: https://wavemax.promo
  • Privacy policy: https://wavemax.promo/privacy-policy
  • Terms of service: https://wavemax.promo/terms-and-conditions
5

Add Environment Variables

Required Environment Variables:
GOOGLE_CLIENT_ID=your_google_client_id_here GOOGLE_CLIENT_SECRET=your_google_client_secret_here

Scopes: The application requests profile and email scopes to get user's basic information.

Facebook OAuth Configuration

Active
1

Create a Facebook App

Visit Facebook Developers and click "Create App".

  • Select "Consumer" as the app type
  • App name: WaveMAX Affiliate Program
  • App contact email: support@wavemax.promo
2

Add Facebook Login Product

In your app dashboard, click "Add Product" and set up "Facebook Login".

3

Configure OAuth Settings

Go to Facebook Login ‚Üí Settings:

  • Client OAuth Login: Yes
  • Web OAuth Login: Yes
  • Valid OAuth Redirect URIs:
    https://wavemax.promo/api/v1/auth/facebook/callback
4

Configure App Domains

In Settings ‚Üí Basic, add your app domains:

wavemax.promo
5

Add Environment Variables

Required Environment Variables:
FACEBOOK_APP_ID=your_facebook_app_id_here FACEBOOK_APP_SECRET=your_facebook_app_secret_here

Instagram OAuth Configuration

Planned
1

Create a Facebook App

Instagram OAuth uses Facebook's infrastructure. Create an app at Facebook Developers.

2

Add Instagram Basic Display

In your Facebook app, add the "Instagram Basic Display" product:

  • Go to Products ‚Üí Add Product
  • Select "Instagram Basic Display"
  • Click "Set Up"

Configure Instagram App

Create an Instagram app within your Facebook app:

  • Display Name: WaveMAX Affiliate Program
  • Valid OAuth Redirect URIs:
    https://wavemax.promo/api/v1/auth/instagram/callback
  • Deauthorize Callback URL: https://wavemax.promo/api/v1/auth/instagram/deauth

Add Test Users

  • Go to Roles ‚Üí Instagram Testers
  • Add Instagram accounts for testing
  • Accept invitations from Instagram accounts

Security Best Practices

  • Always use HTTPS in production
  • Store OAuth credentials in environment variables, never in code
  • Implement state parameter validation to prevent CSRF attacks
  • Regularly rotate client secrets
  • Monitor OAuth usage for suspicious activity
  • Implement proper session management after OAuth login
