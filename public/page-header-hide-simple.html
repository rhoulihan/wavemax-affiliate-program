Simple Script to Hide Page Header

The Problem

The page header with class .page-header.page-header-modern.bg-color-light-scale-1.page-header-sm is still showing when the iframe loads.

Quick Test

First, let's test if the script can find and hide the element. Open the browser console (F12) on the WaveMAX Laundry page and paste this code:

// Test if we can find and hide the page header var header = document.querySelector('.page-header'); if (header) { header.style.display = 'none'; console.log('Page header hidden successfully!'); } else { console.log('Page header not found!'); }

Solution 1: Simple Direct Hide (Recommended)

Add this script right before the closing </body> tag on your page:

<script> // Simple script to hide page header when iframe is present (function() { // Function to hide the page header function hidePageHeader() { var headers = document.querySelectorAll('.page-header, .page-header.page-header-modern'); headers.forEach(function(header) { header.style.display = 'none'; console.log('[WaveMAX] Page header hidden'); }); } // Check if iframe exists function checkAndHide() { var iframe = document.querySelector('iframe[src*="affiliate.wavemax.promo"]'); if (iframe) { hidePageHeader(); return true; } return false; } // Try multiple times with delays if (!checkAndHide()) { var attempts = 0; var interval = setInterval(function() { attempts++; if (checkAndHide() || attempts > 20) { clearInterval(interval); } }, 100); // Check every 100ms for up to 2 seconds } })(); </script>

Solution 2: More Aggressive Approach

If Solution 1 doesn't work, try this more aggressive version:

<script> // Aggressive page header hiding (function() { var style = document.createElement('style'); style.textContent = ` .page-header.page-header-modern, .page-header { display: none !important; } `; document.head.appendChild(style); console.log('[WaveMAX] Added CSS to hide page header'); })(); </script>

Solution 3: jQuery Version (if your site uses jQuery)

If your site uses jQuery, you can use this version:

<script> // jQuery version $(document).ready(function() { if ($('iframe[src*="affiliate.wavemax.promo"]').length > 0) { $('.page-header').hide(); console.log('[WaveMAX] Page header hidden with jQuery'); } }); </script>

Where to Add the Script

  1. Edit your WaveMAX Laundry page HTML
  2. Find where the iframe is embedded
  3. Add one of the scripts above AFTER the iframe code
  4. Save and test
