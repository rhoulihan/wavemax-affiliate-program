WaveMAX Affiliate Registration - Embed Example

This page demonstrates how to embed the WaveMAX affiliate registration form on your website.

Live Example

Integration Code

<!-- Add this iframe to your page where you want the form to appear -->
<iframe 
    id="affiliateFormIframe"
    src="https://wavemax.promo/affiliate-register-embed.html"
    title="WaveMAX Affiliate Registration Form"
    style="width: 100%; border: none; min-height: 800px;"
    scrolling="no">
</iframe>

<!-- Add this script to handle dynamic iframe resizing -->
<script>
// Listen for messages from the iframe
window.addEventListener('message', function(event) {
    // Verify the origin for security
    if (event.origin !== 'https://wavemax.promo') {
        return;
    }
    
    const iframe = document.getElementById('affiliateFormIframe');
    
    switch (event.data.type) {
        case 'form-loaded':
        case 'form-height':
            // Dynamically resize iframe based on content height
            if (event.data.height) {
                iframe.style.height = event.data.height + 'px';
            }
            break;
            
        case 'registration-error':
            // Handle registration errors
            console.error('Registration error:', event.data.message);
            // You can display this error to your users as needed
            break;
    }
});

// Optional: Prefill form data
function prefillForm(data) {
    const iframe = document.getElementById('affiliateFormIframe');
    iframe.contentWindow.postMessage({
        type: 'prefill-form',
        data: data
    }, 'https://wavemax.promo');
}

// Example usage:

</script>

Features

  • Responsive design that adapts to your container width
  • Automatic height adjustment based on form content
  • Secure cross-origin communication
  • Form validation and error handling
  • Redirects to success page in parent window upon completion
  • Optional form prefilling capability

Security Considerations

  • Always verify the origin of messages when using postMessage
  • The form uses HTTPS for all API communications
  • CSRF protection is built into the form
  • Sensitive data is transmitted securely
