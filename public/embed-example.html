Option 4: Responsive Iframe with Auto-Resize

For a responsive iframe that adjusts its height automatically:

<!-- Responsive iframe with auto-resize -->
<div id="affiliate-iframe-container"></div>

<script>
(function() {
    var container = document.getElementById('affiliate-iframe-container');
    var iframe = document.createElement('iframe');
    iframe.src = 'https://wavemax.promo/iframe-embed.html';
    iframe.style.width = '100%';
    iframe.style.border = 'none';
    iframe.frameBorder = '0';
    
    // Auto-resize function
    function resizeIframe() {
        try {
            var height = iframe.contentWindow.document.body.scrollHeight;
            iframe.style.height = height + 'px';
        } catch(e) {
            iframe.style.height = '800px'; // Fallback height
        }
    }
    
    iframe.onload = resizeIframe;
    window.addEventListener('resize', resizeIframe);
    
    container.appendChild(iframe);
})();
</script>

WordPress Shortcode

If the WaveMAX site uses WordPress, you can create a shortcode:

<?php
// Add this to your theme's functions.php file
function wavemax_affiliate_embed_shortcode($atts) {
    $atts = shortcode_atts(array(
        'height' => '800',
        'type' => 'compact'
    ), $atts);
    
    $url = ($atts['type'] === 'full') 
        ? 'https://wavemax.promo/embed-landing.html' 
        : 'https://wavemax.promo/iframe-embed.html';
    
    return '<iframe src="' . $url . '" 
            width="100%" 
            height="' . $atts['height'] . '" 
            frameborder="0" 
            style="border: none;"></iframe>';
}
add_shortcode('wavemax_affiliate', 'wavemax_affiliate_embed_shortcode');

// Usage: [wavemax_affiliate type="compact" height="800"]
?>

Important Notes

  • HTTPS Required: Both sites must use HTTPS for iframe embedding to work properly
  • Mobile Responsive: All embed options are mobile-friendly
  • Tracking: UTM parameters are automatically added to track traffic from embeds
  • Browser Compatibility: Works in all modern browsers (Chrome, Firefox, Safari, Edge)
  • Loading Performance: Consider lazy loading for better page performance
