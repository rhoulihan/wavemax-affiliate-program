API Reference

Constructor Options

Option Type Default Description
size string 'default' Size of the spinner: 'small', 'default', 'large'
speed string 'normal' Animation speed: 'smooth' (3s), 'normal' (2s), 'fast' (1s)
container Element null Container element to append spinner to
message string '' Message to display below spinner
overlay boolean false Show spinner with overlay background
className string '' Additional CSS class names

Methods

.create() - Creates the spinner element
.show() - Shows the spinner
.hide() - Hides the spinner
.updateMessage(message) - Updates the spinner message
.destroy() - Removes the spinner and cleans up

Utility Functions

SwirlSpinnerUtils.showOnButton(button, options)
Shows spinner inside a button and disables it
SwirlSpinnerUtils.showOnForm(form, options)
Shows spinner overlay on a form and disables inputs
SwirlSpinnerUtils.showGlobal(options)
Shows a full-screen global spinner

Integration Guide

1. Include the Script

<script src="/assets/js/swirl-spinner.js"></script>

2. Basic Usage

// Create a spinner instance const spinner = new SwirlSpinner({ container: document.getElementById('my-container'), size: 'default', message: 'Loading...' }); // Show the spinner spinner.show(); // Hide the spinner spinner.hide();

3. Real-World Example: AJAX Form

async function handleFormSubmit(event) { event.preventDefault(); const form = event.target; // Show spinner on form const spinner = SwirlSpinnerUtils.showOnForm(form, { message: 'Processing your request...' }); try { // Make API call const response = await fetch('/api/submit', { method: 'POST', body: new FormData(form) }); if (response.ok) { // Success! showSuccessMessage(); } } catch (error) { showErrorMessage(error); } finally { // Always hide spinner spinner.hide(); } }
