WaveMAX Affiliate Program - Embed Code

This page provides the embed code needed for the wavemaxlaundry.com site to display the customer login page with affiliate tracking.

Embed Code for wavemaxlaundry.com

Add this code to your page at https://www.wavemaxlaundry.com/austin-tx/wavemax-austin-affiliate-program:

<!-- WaveMAX Affiliate Customer Login Embed -->
<div id="wavemax-affiliate-embed">
    <iframe 
        id="wavemax-iframe"
        src="https://wavemax.promo/embed-app.html?route=/customer-login"
        width="100%"
        height="600"
        frameborder="0"
        style="border: none; min-height: 600px;">
    </iframe>
</div>

<script>
(function() {
    // Get affiliate ID from URL
    const urlParams = new URLSearchParams(window.location.search);
    const affiliateId = urlParams.get('affid');
    
    if (affiliateId) {
        // Update iframe src with affiliate ID
        const iframe = document.getElementById('wavemax-iframe');
        iframe.src = `https://wavemax.promo/embed-app.html?route=/customer-login&affid=${affiliateId}`;
    }
    
    // Handle iframe resizing
    window.addEventListener('message', function(event) {
        if (event.origin !== 'https://wavemax.promo') return;
        
        if (event.data.type === 'resize' && event.data.data && event.data.data.height) {
            const iframe = document.getElementById('wavemax-iframe');
            iframe.style.height = event.data.data.height + 'px';
        }
    });
})();
</script>

How It Works

  1. The embed code creates an iframe that loads the WaveMAX customer login page
  2. JavaScript automatically detects the affid parameter from the URL
  3. The affiliate ID is passed to the embedded login page
  4. Customers can either log in or click "Register here" to create a new account
  5. The affiliate ID is preserved throughout the registration process

Example URL

When affiliates share their link:

The embedded iframe will automatically load:
